🧠 Shared Memory — Ring Buffer par PTY
Structure par PTY :
ShmRegion {
  header: ShmHeader { write_seq: AtomicU64, read_seq: AtomicU64, cols: u16, rows: u16 }
  cells: [Cell; cols × rows × 2] // double buffer
  scrollback: RingBuffer<[Cell; cols]> // N lignes d'historique
  dirty_flags: AtomicBitset // quelles lignes ont changé
}
Cell :
Cell { char: u32, fg: Rgb, bg: Rgb, flags: u8 // bold, italic, underline, inverse }
Synchronisation :
daemon écrit cells + incrémente write_seq + set dirty_flags
gui compare read_seq vs write_seq → si différent, lit les lignes dirty → render
Pas de lock — seqlock pattern (writer atomic increment, reader retry on mismatch)
