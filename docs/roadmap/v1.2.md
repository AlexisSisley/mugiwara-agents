## Corrections

### Fix uninstall.sh — parite avec install.sh
- **Probleme** : Le tableau CREW de `uninstall.sh` ne contenait que 12 agents (les originaux). Les 28 agents ajoutes depuis n'etaient pas desinstalles, laissant des agents orphelins dans `~/.claude/skills/`.
- **Solution** : Alignement du tableau CREW de `uninstall.sh` avec celui de `install.sh` (40 agents).
- **Fichier modifie** : `uninstall.sh`

### Fix morgans — disable-model-invocation
- **Probleme** : L'agent `morgans` etait le seul avec `disable-model-invocation: true` dans son front matter YAML. Cela empechait son invocation par les pipelines.
- **Solution** : Passage a `disable-model-invocation: false`.
- **Fichiers modifies** : `skills/morgans/SKILL.md` (source) + `~/.claude/skills/morgans/SKILL.md` (installe)
